# 🧑‍💼 Employee Management System - Full Stack App

A full-stack web application for managing employees. Built using **Spring Boot** (backend) and **React** (frontend) with **MySQL** database integration.

## 🧩 Technologies Used

### Backend (Spring Boot)
- Java 17
- Spring Boot
- Spring Data JPA (Hibernate)
- Spring Web
- Spring Validation
- Lombok
- MySQL
- CORS configuration
- DTO, Exception Handling

### Frontend (React)
- React 18 with JSX
- React Router DOM
- Axios
- Bootstrap 5
- Vite

### Tools
- MySQL Workbench
- Postman
- IntelliJ / VS Code
- Git & GitHub
